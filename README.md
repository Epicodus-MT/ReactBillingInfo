## Setup/Installation Requirements
To run this program:
  * _LOCAL: Go to Terminal_
  * `git clone https://github.com/Epicodus-MT/MTReactJS.git`
  * 'npm install'
  * 'npm run dev'
  * Visit your app at [http://localhost:8080](http://localhost:8080).
